# npole-landing

Minimal static landing page for the npole organization.

## Stack

- Plain HTML
- Plain CSS
- Tiny vanilla JS
- No build step
- No runtime dependencies

## Local development

Open `index.html` directly in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
